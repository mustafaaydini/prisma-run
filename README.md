# 🔮 PRISMA RUN — Across the Prism Universe

An endless runner × color-matching hybrid set in a 5-world universe. Temple Run's speed, Candy Crush's color chains, Monument Valley's pastel minimalism — fully original: every pixel and sound is generated by code, zero external assets.

**Çalıştırmak için:** `index.html`'e çift tıkla. Kurulum yok, internet gerekmez.

## Gameplay
- **← → / A D / swipe** — change lane
- **SPACE / ↑ / tap** — shift prism color (Rose → Cyan → Amber) · **1/2/3** direct select
- **P / Esc** pause · **M** mute
- Match the **gate color** to pass; chain same colors for a growing **COMBO** multiplier
- **Crystals** = +points & shards (◆); matching color = RESONANCE bonus
- **Power-ups:** Shield (S), Magnet (M), Star Frenzy (★ — invincible + smash everything)
- Every ~780 m a **new world**: Dusk Valley → Crystal Caverns → Neon City → Cosmic Void → Solar Dawn, then loops harder & faster (moving sweeper obstacles, double gates...)

## Meta progression
- **Shards (◆):** in-game currency from crystals, missions, daily login
- **Shop:** 5 skins (Prism, Comet Orb, Stardart, Nova Star, Crown Jewel) + 4 upgrades (Starting Shield, Magnet/Star Boost, Combo Guard)
- **Missions:** 3 rotating goals with tiered rewards, tracked across runs
- **Daily reward**, persistent best score / stats (localStorage)

## 💰 Para kazanma (monetization) rehberi

Oyunda gelir altyapısı hazır; gerçek paraya çevirmek için tek yapman gereken aşağıdaki entegrasyonlardan birini bağlamak:

### 1. Ödüllü reklam (en kolay ve en etkili yol)
Oyunda iki ödüllü reklam noktası **çalışır durumda** (şu an 3 saniyelik demo reklam oynatıyor):
- **Revive** — ölünce "watch ad to revive"
- **2x Shards** — oyun sonunda ödülü ikiye katla

`index.html` içindeki `Ads.show()` fonksiyonuna (dosyanın başında, yorumlarla işaretli) portalın SDK çağrısını koy:
- **CrazyGames** → `window.CrazyGames.SDK.ad.requestAd('rewarded', {adFinished: onReward})`
- **Poki** → `PokiSDK.rewardedBreak().then(ok => ok && onReward())`
- **GameMonetize / GameDistribution** benzer şekilde

Bu portallara oyunu **ücretsiz yüklüyorsun**, onlar reklam gösterip geliri seninle paylaşıyor. Tek dosyalık HTML5 oyunlar tam da bu platformların istediği format.

### 2. Shard paketi satışı (IAP)
Mağazadaki **SHARDS sekmesi** hazır: 3 paket ($1.99 / $4.99 / $9.99). Kodda `PAYMENT_LINKS` sabitine kendi ödeme linklerini koy (Stripe Payment Link, Gumroad, itch.io ürün linki). Link boşken buton kullanıcıya "yapılandırılmadı" uyarısı gösterir — yani şu haliyle de güvenli.

### 3. Yayınlama kanalları
| Kanal | Model |
|---|---|
| **Poki / CrazyGames** | Reklam gelir paylaşımı (en yüksek trafik) |
| **itch.io** | "İsteğe bağlı öde" veya sabit fiyat + %90'a kadar pay |
| **Kendi siten + AdSense** | Sayfa reklamları (oyun tek dosya, host etmesi bedava: GitHub Pages) |
| **Google Play (TWA/Capacitor ile sar)** | AdMob ödüllü reklam + IAP |

**Önerilen ilk adım:** Oyunu CrazyGames'e gönder (developer.crazygames.com) — inceleme birkaç gün sürer, SDK entegrasyonu `Ads.show()` içine 3 satır.

## 📱 Google Play'de yayınlama (Android)

Oyun artık gerçek bir PWA (Progressive Web App): `manifest.webmanifest`, `sw.js` (offline çalışma) ve 3 ikon dosyası (`icon-192.png`, `icon-512.png`, `icon-maskable-512.png`) hazır — bunlar bir Android uygulamasının **teknik ön koşulu**. Bu makinede Java/Android SDK kurulu olmadığından ve Play Store yayını senin kendi Google hesabınla yapılması gerektiğinden (bunları senin adına oluşturamam), kalan adımlar şöyle — hiçbiri Android Studio gerektirmiyor:

1. **Oyunu herkese açık bir HTTPS adresinde barındır** (TWA/PWA paketleyicileri için şart). En hızlısı:
   - **GitHub Pages** — bu klasörü bir GitHub reposuna yükle, Settings → Pages'ten yayınla (ücretsiz).
   - veya **Netlify/Vercel** — `game1` klasörünü sürükle-bırak ile deploy et (hesap açman yeterli, dakikalar sürer).
2. **PWABuilder.com'a git**, yayındaki URL'ni gir. Site otomatik olarak manifest/service worker'ı okur ve **imzalanmış Android App Bundle (.aab)** üretir — hiçbir kurulum gerekmez, tamamen tarayıcıda.
3. **Google Play Console** hesabı aç (tek seferlik 25$), yeni uygulama oluştur, PWABuilder'ın verdiği `.aab` dosyasını yükle, mağaza bilgilerini (açıklama, ekran görüntüleri, ikon — `icon-512.png` hazır) doldur ve incelemeye gönder.

Alternatif (daha teknik, tam kontrol istersen): `npx @bubblewrap/cli init --manifest=https://senin-adresin/manifest.webmanifest` komutu bir Android Studio Gradle projesi oluşturur; ilk çalıştırmada JDK+Android SDK indirmeni ister (birkaç GB).
